# 📌 Menu de Navegação Animado - HTML & CSS

Projeto de desenvolvimento de um **menu de navegação com efeito animado no hover**, utilizando apenas HTML5 e CSS3.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts (Poppins)

---

## 🎯 Objetivo

Praticar conceitos de estilização e interatividade com CSS, incluindo:

* Criação de menus de navegação
* Uso de pseudo-elementos (`::after`)
* Animações com `transition`
* Efeitos visuais com hover

---

## 🧩 Funcionalidades

* Menu horizontal centralizado
* Itens com efeito de sublinhado animado
* Cores personalizadas para cada item
* Animação suave ao passar o mouse

---

## ✨ Diferenciais técnicos

* Uso de pseudo-elementos para criar animações sem JavaScript
* Aplicação de `nth-child` para estilização individual dos itens
* Transições suaves com CSS
* Código limpo e organizado
